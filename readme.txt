Instructions to Run 
* Install Scikit-learn , matplotlib, panda and numpy.
* Unzip the zip file and open the folder in jupyter. 
* Open gd.ipynb file and run. 
* You will see the different parameter values ie, error, entropy error etc.. on the console. Wait till done is printed on the console.
* Once done, 4 csv files will be generated in the same folder. 
 - PredictionsWithTenThousand.csv is the predicted output with ten thousand iterations.
 - PredictionsWithHundredThousand.csv is the predicted output with 1 lakh iterations.
 - PredictionsWithMillion.csv is the predicted output file with 1 million iterations.
 - PredictionOnTestFromLib.csv is the output predicted file from sklearn logistic regression model.
